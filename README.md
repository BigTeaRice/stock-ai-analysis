# AI股票分析系統

使用AI技術分析港股和美股，包含多種機器學習模型和技術指標。

## 功能特點

- 📊 **多模型AI預測**: ARIMA、LSTM、線性回歸
- 📈 **技術指標分析**: ATR、布林帶、RSI、MACD
- 🎯 **交易信號生成**: 自動識別買入/賣出信號
- 📱 **響應式設計**: 支持手機和電腦瀏覽
- 🔄 **實時數據**: 從Yahoo Finance獲取最新數據
- 📊 **可視化圖表**: 交互式圖表展示
